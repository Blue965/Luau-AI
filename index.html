Plugin Luau AI pour Roblox Studio

Connecte ton Roblox Studio directement a Luau AI. Genere une cle de session, colle-la dans le plugin et code sans quitter Studio.

Cle de session
Ta cle de session permet au plugin Roblox Studio de s'identifier aupres de Luau AI. Elle est liee a ton compte Roblox et expire apres 24h. Generes-en une nouvelle a chaque session Studio.
— Aucune cle generee —
Activer la connexion dans Studio
Dans le plugin Roblox Studio, colle ta cle de session dans le champ prevu et clique Activer. Luau AI sera alors directement connecte a ton jeu.

Installation rapide

1
Generer ta cle
Clique sur Generer une cle de session ci-dessus. Copie-la.
2
Telecharger le plugin
Telecharge LuauAI.lua et installe dans Plugins → Plugin Manager.
3
Coller la cle dans Studio
Dans le plugin Studio, colle ta cle dans le champ Cle de session et clique Activer.

C'est pret !
