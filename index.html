Plugin Luau AI pour Roblox Studio

Connecte ton Roblox Studio directement a Luau AI. Demande de l'aide sur ton code Luau sans quitter Studio.

Telecharger le Plugin Studio
Fichier .rbxm a installer dans Roblox Studio — Plugin Manager → Install from file.
Compatible Roblox Studio 2024+

Comment installer

1
Telecharger le plugin
Clique sur le bouton ci-dessus pour telecharger le fichier LuauAI.lua.
2
Ouvrir Roblox Studio
Dans Studio, va dans Plugins → Plugin Manager → Install from file et selectionne le fichier telecharge.
3
Lancer le plugin
Un bouton Luau AI apparait dans la barre de plugins. Clique dessus pour ouvrir le panneau.
4
Poser ta question
Le plugin envoie tes questions directement a luau-ai.vercel.app/api/chat et affiche la reponse dans Studio.
